Five Quadrant Image Centering

Considering that the user can not validate the position of the camera, and will need feedback to position the camra correctly, Appliance Sight implements a unique '5 quadrant image centering' algorithm. A virtual grid with five quadrants is placed over the captured image. The white pixel count is taken for each quadrant (after filtering). If any outer quadrant has the most pixels, the user is signaled and/or the image is programatically centered.

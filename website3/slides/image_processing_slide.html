Image Processing


Collecting and preparing data to train the model.

  • Image captured by Raspberry Pi

    We start with about 300 images of the keypad captured from the users fingertip camera. Each greyscale image is 128 x 128 pixels in size

  • Five Quadrant Image Centering

    The pixels are centered in the screen using our Five Quadrant Image Centering Algorithm

  • Augmenting the data set

    To make the training data more robust, we apply the following filters to each image:

    • Affine Transformation
    • Gaussian Noise
    • Contrast Normalization
    • Median Blur
    • Gaussian Blur

  • Image Centering

    ~6000 images

  • Determine Threshold

    Most of the data is noise. Data at the 95th percentile is useful for classification

  • Resize image

    Resize images to 64x64 and convert to greyscale. We then apply median blur

  • Image Threshold 64x64

  • ~ 6000 Images
    ready to train the model

Integration

Sequence of steps to capture a button image and report to user.

  • Image Capture

    User captures an image from fingertip camera

  • Image Processing

    Resolution is changed, Greyscale and median bluring is applied. Threshold values are collected.

  • Positioning Guidance

    5 Quadrant image centering is applied to determine where the central mass of the index lies. The user is given feedback to move the camera.

  • Classification

    Convolutional Neural Network conducts Optical Character Recognition

  • Audio Feedback

    The button identified by the neural network is announced to the user
